# MA-DR-VRPTW
多Agent协同的动态车辆路径优化系统（MA-DR-VRPTW）
本项目构建了一套面向复杂物流场景的多Agent协同动态调度系统，融合车辆路径优化问题（VRPTW）建模、多车辆协同调度与时间窗约束处理机制。系统由需求建模Agent、车队管理Agent、路径优化Agent（基于改进蚁群算法）、环境感知Agent及动态重规划Agent组成，实现了复杂约束下的智能决策闭环。

在运行过程中，系统可根据订单需求、车辆容量及时间窗约束生成初始最优路径，并结合环境Agent实时感知交通与天气扰动，当外部变化超过阈值时，自动触发重规划机制，实现路径的自适应调整。

实验结果表明，相较传统静态调度方法，系统在多约束条件下可将配送成本降低约28%，调度响应速度提升约45%，显著增强了复杂环境下的鲁棒性与智能化水平。
